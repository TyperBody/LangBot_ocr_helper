# LangBot_ocr_helper

这是一个 LangBot OCR 助手插件。

[![开发群聊](https://img.shields.io/badge/开发群聊-472263434-blue)](https://qm.qq.com/cgi-bin/qm/qr?k=472263434) ![LangBot版本](https://img.shields.io/badge/LangBot-4.3+-blue) ![Python版本](https://img.shields.io/badge/Python-3.10+-blue)

功能说明

此工作区包含一个 OCR 助手插件，能够：
- 监听个人和群聊中的消息
- 自动检测消息中的图像
- 使用 EasyOCR 库识别图像中的文本
- 将识别结果返回给用户

开发说明

插件使用 LangBot 插件 SDK 开发，遵循插件开发规范。
